**national-awards**

A web-based nomination and award management system for the **Pride of India National Awards – Chennai 2026** organized by **Global Achievers Foundation** and **G & G Group of Institutions**.

This repository contains the landing page and automation scripts used to manage nominations, payment processing, and communication with nominees.

The system integrates multiple services to streamline the award nomination process including:

• **GitHub Pages / Website Hosting** – Event landing page and information portal
• **Google Forms** – Online nomination and evaluation form
• **Google Apps Script** – Backend automation and workflow management
• **Razorpay** – Secure payment processing for nomination contributions
• **Gupshup WhatsApp API** – Automated WhatsApp notifications
• **Google Sheets** – Central database for nomination tracking

### Key Features

• Event landing page with full award details
• Online nomination and evaluation form
• Automatic application submission confirmation
• Email notification to nominees and administrators
• WhatsApp notification using Gupshup API
• Razorpay payment link integration
• Payment confirmation via Razorpay webhook
• Automated email and WhatsApp acknowledgement after payment
• Nomination tracking via Google Sheets
• Scalable structure for future national and international award events
